## 개요

`local-html-editor`는 로컬 HTML 파일을 브라우저에서 직접 편집하게 해주는 Chrome 확장 프로그램입니다. Chrome에서 `.html` 파일을 열고 편집 모드로 들어가면 텍스트, 서식, 링크, 이미지를 페이지 위에서 바로 수정할 수 있습니다.

AI가 만든 단일 HTML, 랜딩 페이지, 뉴스레터, 이력서, 프로토타입, 브랜드 초안을 고칠 때 유용합니다.

## 주요 기능

- `file://` 로컬 HTML 시각 편집.
- Notion 스타일 아이콘 툴바와 호버 툴팁.
- 4-128 글자 크기, 줄 간격, 밑줄, 강조, 색상, 정렬, 링크.
- 이미지 삽입, 교체, 크기 조절, 클립보드 붙여넣기.
- 색상 스와치와 RGB 선택기.
- 서식 복사: 전체, 텍스트 스타일, 색상, 강조, 정렬.
- 자동 저장과 파일별 기록.
- 브라우저 언어 기반 다국어 UI.
- 브라우저 테마 기반 라이트/다크 모드.

## Install

1. Open `chrome://extensions`.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this project folder.
5. Enable **Allow access to file URLs**.

## Release ZIP

```text
dist/local-html-editor-extension.zip
```
